Descripción
La API de Netflix proporciona acceso a información sobre películas y programas de TV disponibles en la plataforma de streaming de Netflix. Permite a los usuarios obtener listas de películas y programas de TV, así como detalles específicos sobre cada título, como el elenco, el director, el país de producción, la fecha de lanzamiento y más.

Base URL
El punto de acceso base para la API de Netflix es: http://localhost:3000

Paths Disponibles
Obtener Todas las Películas
http
Copy code
GET /api/movies
Este endpoint devuelve una lista de todas las películas disponibles en Netflix.

Respuestas:
200 OK: La solicitud fue exitosa y se devuelve una lista de películas.

Obtener Detalles de una Película Específica
http
Copy code
GET /api/movies/{show_id}
Este endpoint devuelve los detalles de una película específica identificada por su show_id.

Parámetros de Ruta:
show_id (string): Identificador único de la película.
Respuestas:
200 OK: La solicitud fue exitosa y se devuelve los detalles de la película.
json
Copy code
{
  "show_id": "string",
  "type": "string",
  "title": "string",
  "director": "string",
  "cast": "string",
  "country": "string",
  "date_added": "string",
  "release_year": "integer",
  "rating": "string",
  "duration": "integer",
  "listed_in": "string",
  "description": "string"
}
404 Not Found: La película no fue encontrada.
